# Visit Hair Clinic

A premium, fully responsive website for **Visit Hair Clinic** — a leading hair transplant clinic based in Istanbul, Turkey. The site showcases the clinic's procedures, results, and expertise with smooth animations and multi-language support.

![Screenshot](screenshot.png)

## Live Demo

[https://safaberk334.github.io/visit-hair-clinic/](https://safaberk334.github.io/visit-hair-clinic/)

## Features

- **Multi-Language Support** — Turkish, English, and Arabic
- **Before / After Slider** — Interactive image comparison showcasing real patient results
- **Animated Statistics Counter** — Eye-catching numbers that count up on scroll
- **Procedure Showcases** — Detailed sections for FUE, DHI, and Sapphire FUE techniques
- **Patient Testimonials** — Real reviews from satisfied patients
- **WhatsApp Integration** — One-tap contact button for instant communication
- **Responsive Design** — Optimized for all screen sizes and devices
- **Smooth Scroll Animations** — Powered by AOS (Animate on Scroll)
